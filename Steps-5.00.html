How to Unpivot a pivoted data in a query ? 5

 Unpivot Performers almost the reverse operation of Pivot. Using the same database record Purchase record table as an example, suppose the table produced in the example step 2 is stored in the database as pivot and you want to unpivot the table.  To do that by rotate the column identifiers into row values.  The query looks like this.

First, Create Pivot table, Copy the below code to create pivot table. Then follow the next prompt

SELECT CustomerName, Expenses, PurchaseAmount
FROM [dbo].[UnpivotPurchaseRec]
UNPIVOT
(
 PurchaseAmount
 FOR Expenses IN (OtherExpenses, Shoes, Cloth,BeautyProducts) 
 ) AS UnPivotTable
