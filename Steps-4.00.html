How to take data that is formatted for analysis and Pivot it ? 4

Using the Purchase record   table sample data to determine the total expenses and total amount of each Product Name by Customer name and Customer ID, the following query Provides this report.

The pivot record set should be look like the below screenshot.

SELECT [CustomerName],		           --- ------------------------Grouping columns 
		[CustomerID],
		SUM(OtherExpenses) AS OtherExpenses ,		
		SUM(ISNULL(Shoes,0)) AS Shoes, 
		SUM(ISNUll(Cloth, 0)) AS Cloth,
		SUM(ISNULL(BeautyProducts,0)) AS BeautyProducts		-----------------Spreading Columns 
FROM
(
SELECT	[CustomerName],
		[CustomerID],
		[OtherExpenses],
		[ProductName],
		[Amount]		---------------------------------Aggregated Columns 
FROM [Practice].[dbo].[PurchaseRecord]
)	AS PivotData
Pivot
(
SUM(Amount) for ProductName in 
(Shoes,Cloth, BeautyProducts))	AS Pivoting 
GROUP BY CustomerName, CustomerID



Output :


In above query we calculate the sum of  all product names and other expenses corresponding by the Transaction date, Customer Name, and CustomerID.
